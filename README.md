# Seasonality in Bond Trading: Exploring Calendar-Based Trends & Patterns

This repository contains a Jupyter Notebook for analyzing bond data. The goal is to provide insights and visualizations to help understand bond market trends and performance.

## Contents

- `bond_trading_small`: A small CSV of first few lines of the data file.
- `bond_analysis`: Jupyter notebooks analysis & visualizations.

## Miscellaneous
